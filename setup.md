# LockIn Demo — Setup

No Supabase or database. Front-end only.

1. **Install and run**
   ```bash
   npm install
   npm run dev
   ```
2. Open **http://localhost:3000** and click **Open demo room**.
3. Use **Simulate** buttons to switch focus state (On task / Warning / Locked / Break) and test the warning banner and lock overlay.
4. Use the chat to send messages (stored in memory until refresh).

Optional: load the Chrome extension from the `extension/` folder to test domain-based focus state (blocked sites → lock).
